Overview

The Game

LingoLink is a browser-based strategy game built around the intersection of world geography and linguistics. Players begin from a randomly assigned country and must chain connections to other countries through shared languages, building streaks to accumulate points.

Mechanics

How It Works

The scoring system rewards strategic thinking alongside raw knowledge. Common languages like English or Spanish are widely available but yield low returns, while rarer languages like Quechua or Fulfulde carry significantly higher point values.

Overusing any single language locks it out, forcing players to draw on the full breadth of their linguistic knowledge.

Technical

How It's Built

Built with Next.js and TypeScript, the game logic was originally written in Java and ported to the web with a redesigned UI. The underlying dataset covers 198 countries and 285 languages, with rarity scores derived from real-world co-occurrence data.

Background

Origin & Iteration

LingoLink was developed as a team project, motivated by a shared interest in travel and the role language plays in connecting people across borders. Since the first release of the game, I've continued to iterate on the original command-line based game and designed the user interface.

Please give this game a go and let me know of any improvements you have! I'm always curious if I've missed any languages or countries, as the world is so big.
